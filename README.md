# CalculatorCLI

A simple C# console calculator.

## How to Run

```bash
dotnet run
```

## Features

- Add, subtract, multiply, and divide numbers
- Enter multiple numbers separated by commas or spaces
- Handles division by zero errors

## Example

```
Enter numbers: 10, 20, 30
Addition: 10 20 30 = 60
```

## Files

- `Program.cs` - Main program
- `Classes/Calculator.cs` - Calculator operations
- `FLOWCHART.md` - Program flow
- `PSEUDOCODE.md` - Algorithm
